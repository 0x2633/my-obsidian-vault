**均值回歸 (Mean Reversion)** 是一種金融理論，假設價格（或報酬率、波動率）在==偏離其歷史平均水平（Mean）後，最終都有回到該平均水平的傾向==。

這在統計學上與「向平均回歸」的概念一致：當極端事件發生後，後續的數據通常會趨於正常。

---

## 1. 核心邏輯：橡皮筋效應

您可以將價格與均線的關係想像成一條**橡皮筋**：

- 當橡皮筋被拉得太長（價格暴漲或暴跌），張力會增加。
- 當張力達到極限（統計上的高標準差區域，如布林通道 $\pm 3\sigma$），價格往均線方向彈回的機率就會大幅提升。

---

## 2. 判斷均值回歸的技術指標

在實作您的交易策略時，通常使用以下工具來量化「偏離程度」：

|**工具**|**判斷邏輯**|
|---|---|
|**乖離率 (BIAS)**|計算當前價格與均線的百分比差距。|
|**布林通道 (BB)**|價格觸及上下軌後，預期會回到中軌（20 SMA）。|
|**RSI 指標**|超買（>70）或超賣（<30）區間常伴隨均值回歸。|
|**Z-Score**|衡量價格偏離均值多少個標準差，是量化交易最常用的指標。|

---

## 3. 交易策略實作 (以 TSM/ZN 為例)

均值回歸策略在==**盤整市場（Ranging Market）**==最為有效，但在強勢趨勢中則具備較高風險（即「接刀子」）。

### 策略逻辑（JSON Specification 範例）

JSON

```
{
  "strategy_name": "Mean Reversion - Bollinger",
  "asset": ["TSM", "ZN"],
  "entry_conditions": {
    "long": "Price < (SMA20 - 2 * StdDev)",
    "short": "Price > (SMA20 + 2 * StdDev)"
  },
  "exit_target": "Price touches SMA20",
  "stop_loss": "Price < (SMA20 - 3 * StdDev)"
}
```

---

## 4. 配對交易 (Pairs Trading)：進階均值回歸

可以計算兩個高度相關資產（例如：TSM 與 SOXX 半導體 ETF）的**價差（Spread）**。

1. **邏輯：** 當兩者價差擴大到異常水平（High Z-Score），賣出高估者，買入低估者。
2. **預期：** 價差最終會收斂回歸至平均值。

---

## 5. 技術面的潛在陷阱

- **趨勢陷阱 (Trend Risk)：** 如果市場出現基本面重大改變（如 TSM 營收腰斬），價格會沿著邊軌持續下跌，這時均值回歸會導致連續虧損。
- **計算延遲：** 在處理即時數據流時，若計算窗口過大（$N$ 過大），回歸信號會過於滯後。
